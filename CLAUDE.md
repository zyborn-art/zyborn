# ZYBORN ART - Main Website

## Overview
Canned BTC landing page with CMS and email capture system.

## Stack
- Static site generator: Eleventy (11ty)
- Hosting: Vercel
- Frontend: HTML/CSS/JS with Nunjucks templates
- CMS: File-based (Markdown + JSON in /content/)

## Architecture
- /content/ - CMS-managed content (pages, settings)
- /src/ - Eleventy templates (.njk)
- /public/ - Static assets
- /api/ - Vercel serverless functions
- /shared/ - Shared utilities
